# Welcome to EMI For Create Stock
## What's this ?
This is a mod making you craft in Create Stock easier with Fill Recipe support. It could automatically search in your inventory and stock inventory finding crafting stuff. Only click fill recipe button, stock will search stuff and send it automatically (could turn off by config).
## Others
This mod is developed for my use, and I just make it public if there are others need. After I finish all function, it will nearly not update. And I have a little time for coding, so even now, it may not update very fast.
